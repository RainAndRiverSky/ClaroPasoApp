# Feature: The "Persistent Audio Helper" (The Ear Icon)

Since the user shouldn't be annoyed by repeated pop-ups, we use an Invisible Teacher approach:

The "Ear" Icon: A small, friendly ear icon stays in the corner.

Function: If the user is staring at a screen for more than 10 seconds without moving, the "Ear" pulses.

The Interaction: If they tap it, the AI says: "¿Te quedaste trabada? No pasa nada. Estás en la sección de la escuela. Aquí puedes ver los grados del niño o subir sus vacunas. ¿Qué quieres hacer?"
